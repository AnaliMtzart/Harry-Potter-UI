# HARRY POTTER UI

## 1. Resumen del proyecto

Proyecto donde se **construyó una _página web_ para visualizar un
_conjunto (set) de datos_** que se adecúa a lo que el usuario necesita.

Es una página web que permite 
**visualizar la data y filtrarla por estudiantes, staff y personajes finados**. 

Data utilizada:

* [Harry Potter](hp-ui/src/data/hp-characters.json):
  En este set encontrarás una lista con 25 personajes con sus respectivos datos.

### 3. Instrucciones

* Abrir consola:
    - Clonar el repositorio de manera local
    - Instalar paquetes con: npm install
    - Desde la consola hacer la ruta a la carpeta Harry-Potter-UI
    - Ingresar: json-server --watch hp-characters.json --port 3004
        (para poder agregar un nuevo personaje)  
        
* Desde VS/Editor de código:
    - Abrir el proyecto en VS o en otro editor de código
    - Abrir nueva terminal
    - Acceder a la carpeta hp-ui
    - Ingresar en terminal: npm start
    
* Desde el navegador ya se podrá visualizar la app/interfaz
